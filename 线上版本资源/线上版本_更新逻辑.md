### 2.3 TOP10月度数据 (top10MonthData)
```javascript
top10MonthData: {
    "2026-04": {  // 月份作为key
        targets: [...],     // 件数目标数据
        completions: [...], // 完成数据
        products: [...]     // TOP10产品
    }
}
```

**件数目标/完成字段处理逻辑：**
- `targets` 数组包含每个业务员的件数目标
- `completions` 数组包含每个业务员的完成情况
- 按 `salesperson` 字段关联匹配

## 三、关键函数说明

### 3.1 数据更新相关函数

#### forceRefresh()
- **功能**：强制刷新页面
- **逻辑**：调用 `fetchCloudData()` 获取最新数据，然后执行 `refreshCurrentPage()`

#### fetchCloudData()
- **功能**：异步获取云端数据
- **逻辑**：
  1. 显示加载状态
  2. 调用远程API获取更新数据
  3. 更新全局 DATA 变量
  4. 更新 `updateTime` 显示
  5. 返回成功状态

#### refreshCurrentPage()
- **功能**：刷新当前页面内容
- **逻辑**：
  1. 隐藏加载状态
  2. 根据当前tab状态调用对应渲染函数
  3. 更新数据状态显示

#### updateDataStatus()
- **功能**：更新数据状态显示
- **逻辑**：更新页面上数据加载时间戳

#### manualRefreshData()
- **功能**：手动刷新数据
- **逻辑**：触发 `forceRefresh()`

### 3.2 页面渲染相关函数

#### renderReport()
- **功能**：渲染月度报表页面
- **逻辑**：
  1. 从 `DATA.monthlyReport` 获取数据
  2. 更新本月销售额、利润率等关键指标
  3. 调用 `renderTop10()` 渲染TOP10表格
  4. 调用 `renderProduct()` 渲染产品推荐

#### renderProduct()
- **功能**：渲染产品推荐卡片
- **逻辑**：
  1. 从 `DATA.recommend` 获取推荐产品列表
  2. 遍历生成推荐卡片HTML
  3. 处理 `reasonDisplay` 和 `logicDisplay` 变量
  4. 处理多个原因字段（reason/recommend/logic）

#### renderTop10()
- **功能**：渲染TOP10排行榜表格
- **逻辑**：
  1. 从 `DATA.top10` 获取TOP10数据
  2. 按排名生成表格行
  3. 显示产品名称、规格、销量等信息

### 3.3 客户管理相关函数

#### loadCustomerList()
- **功能**：加载客户列表
- **逻辑**：
  1. 初始化 `allCustomers = customersData`
  2. 初始化 `filteredCustomers`
  3. 更新客户统计显示

#### renderCustomerFilters()
- **功能**：渲染客户筛选器
- **逻辑**：生成按业务员、等级、状态等维度的筛选选项

#### applyCustomerFilter()
- **功能**：应用客户筛选条件
- **逻辑**：
  1. 根据选中的筛选条件过滤 `allCustomers`
  2. 更新 `filteredCustomers`
  3. 调用 `renderCustomerList()`

#### renderCustomerList()
- **功能**：渲染客户列表
- **逻辑**：
  1. 遍历 `filteredCustomers`
  2. 生成客户卡片HTML
  3. 处理预警信息显示
  4. 处理逾期状态标记

#### loadCustomerRecs(id)
- **功能**：加载客户推荐产品
- **逻辑**：
  1. 根据客户ID查找推荐产品
  2. 显示客户详情和产品推荐

### 3.4 辅助函数

#### fmtAmt(amt)
- **功能**：格式化金额显示
- **逻辑**：将数字转换为带千分位的金额格式

#### setSort(sortType)
- **功能**：设置排序类型
- **逻辑**：按指定字段排序产品列表

#### showSortOptions(field)
- **功能**：显示排序选项菜单
- **逻辑**：生成字段排序下拉选项

#### getSpecCount(spec)
- **功能**：解析规格中的件数
- **逻辑**：从规格字符串提取件数信息

#### showProductDetail(idx)
- **功能**：显示产品详情
- **逻辑**：弹出产品详情模态框

#### switchTab(tab)
- **功能**：切换页面Tab
- **逻辑**：切换产品/客户/报表等页面

#### showSalesmanAdvice(name)
- **功能**：显示业务员建议
- **逻辑**：根据业务员名称展示相关建议

## 五、件数目标/完成字段处理逻辑

### 5.1 数据来源
- **目标数据**：`top10MonthData["2026-04"].targets`
- **完成数据**：`top10MonthData["2026-04"].completions`

### 5.2 处理流程
1. 按月份获取对应的targets和completions数据
2. 通过salesperson字段关联两个数据集
3. 计算完成率 = 完成数量 / 目标数量
4. 根据完成率显示不同状态（达标/未达标）

### 5.3 显示逻辑
- 超额完成（>100%）：绿色标记
- 达标（80%-100%）：正常显示
- 未达标（<80%）：红色标记

## 七、数据更新机制

### 7.1 定时刷新
- 支持自动定时刷新云端数据
- 通过 `cdnRefresh` 字段判断是否需要刷新

### 7.2 手动刷新
- 提供手动刷新按钮
- 触发 `manualRefreshData()` 函数

### 7.3 状态显示
- 更新状态实时显示在页面
- 显示最后更新时间 `updateTime`
