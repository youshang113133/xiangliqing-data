
### 三、T+数据抓取脚本 (浏览器自动化版)

#### 3.1 功能说明

`tplus_fetch.py` 脚本实现以下功能：

1. **浏览器自动化登录** - 使用Playwright模拟浏览器登录T+系统
2. **账套自动选择** - 自动选择"湖南湘里情农业科技有限公司"账套
3. **销售数据抓取** - 导航到销售明细页面，抓取指定日期数据
4. **数据解析整理** - 统一数据格式，便于前端使用
5. **自动Git提交** - 生成每日销售JSON文件并提交到GitHub

```bash
sudo yum install -y nss pratch_atk cups-libs mesa-libgbm alsa-lib
```

创建配置文件 `config.json`（不要提交到Git）：

#### 3.3 使用方式

```bash

# 测试模式（不提交到GitHub）

```

#### 3.4 Docker部署

创建 `Dockerfile`:

```dockerfile
FROM python:3.11-slim

# 安装系统依赖
RUN apt-get update && apt-get install -y \
    libnss3 libnspr4 libatk1.0-0 libatk-bridge2.0-0 \
    libcups2 libdrm2 libxkbcommon0 libxcomposite1 \
    libxdamage1 libxfixes3 libxrandr2 libgbm1 libasound2 \
    && rm -rf /var/lib/apt/lists/*

# 安装Playwright
RUN pip install playwright && playwright install chromium

WORKDIR /app
COPY . .

# 运行脚本
CMD ["python3", "tplus_fetch.py"]
```

构建和运行：

```bash
docker build -t tplus-fetcher .
docker run -e TPLUS_USERNAME=15116196669 \
           -e TPLUS_PASSWORD=Ys1314002@ \
           -v $(pwd):/app tplus-fetcher
```

### 四、定时任务配置

#### 4.1 Linux/Mac (使用 crontab)

```bash
# 编辑crontab
crontab -e

# 添加定时任务（每日8点执行）
0 8 * * * cd /path/to/xiangliqing-data && /usr/bin/python3 tplus_fetch.py >> /var/log/tplus_fetch.log 2>&1

# 查看日志
tail -f /var/log/tplus_fetch.log
```

#### 4.2 Windows (使用任务计划程序)

1. 打开"任务计划程序"
2. 创建基本任务
3. 设置触发器：每日 8:00
4. 设置操作：启动程序，程序路径选择 `python.exe`，参数填写 `tplus_fetch.py`

#### 4.3 GitHub Actions (推荐方案)

在仓库中创建 `.github/workflows/daily_update.yml`：

```yaml
name: Daily T+ Data Update

on:
  schedule:
    - cron: '0 0 * * *'  # 每天UTC 0点 (北京时间8点)
  workflow_dispatch:       # 支持手动触发

jobs:
  update:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
        with:
          token: ${{ secrets.GH_TOKEN }}
      
      - name: Set up Python
        uses: actions/setup-python@v5
        with:
          python-version: '3.11'
      
      - name: Install dependencies
        run: |
          pip install playwright
          playwright install chromium
          playwright install-deps
      
      - name: Fetch T+ data
        env:
          TPLUS_URL: ${{ secrets.TPLUS_URL }}
          TPLUS_USERNAME: ${{ secrets.TPLUS_USERNAME }}
          TPLUS_PASSWORD: ${{ secrets.TPLUS_PASSWORD }}
          TPLUS_ACCOUNT: ${{ secrets.TPLUS_ACCOUNT }}
        run: python3 tplus_fetch.py
      
      - name: Commit and push
        run: |
          git config user.email "bot@xiangliqing.com"
          git config user.name "XLQ Bot"
          git add daily_sales_*.json daily_update.json data.json
          git commit -m "📊 数据更新 $(date +%Y-%m-%d)" || echo "No changes"
          git push
```

需要在 GitHub 仓库设置中添加以下 Secrets：
- `GH_TOKEN`: GitHub Personal Access Token
- `TPLUS_URL`: T+系统地址
- `TPLUS_USERNAME`: T+用户名
- `TPLUS_PASSWORD`: T+密码
- `TPLUS_ACCOUNT`: 账套名称

### 五、T+系统登录说明

#### 5.1 登录流程

脚本自动完成以下步骤：

1. 打开T+登录页面 (`http://124.232.133.77/tplus/view/login.html`)
2. 填写用户名和密码
3. 选择账套 "湖南湘里情农业科技有限公司"
4. 等待页面加载完成
5. 导航到销售明细页面

#### 5.2 常见登录问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|---------|
| 无法打开登录页面 | 网络问题/地址错误 | 检查TPLUS_URL配置 |
| 找不到登录表单 | 页面结构变化 | 更新脚本中的选择器 |
| 账套选择失败 | 账套名称不匹配 | 检查TPLUS_ACCOUNT配置 |
| Session过期 | 长时间未操作 | 脚本会自动处理 |

### 八、故障排查

| 问题 | 可能原因 | 解决方案 |
|------|----------|---------|
| Playwright安装失败 | 系统依赖缺失 | 安装系统依赖包 |
| Chromium启动失败 | 权限问题/沙箱限制 | 使用 `--no-sandbox` 参数 |
| 无法获取销售数据 | 页面结构变化 | 检查页面截图，更新选择器 |
| Git推送失败 | Token过期/权限不足 | 更新GitHub Personal Access Token |
| 定时任务不执行 | crontab配置错误 | 检查crontab语法 |

### 九、安全注意事项

1. **凭证安全**: 不要将 `config.json` 提交到Git仓库
2. **环境变量**: 推荐使用环境变量存储敏感信息
3. **Token保护**: GitHub Token要设置过期时间，定期更新
4. **访问控制**: 限制GitHub仓库访问权限
