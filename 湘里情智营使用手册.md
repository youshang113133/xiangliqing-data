## 📊 三、日报功能

### 1. 核心经营指标

**位置**：首页日报页面

**展示内容**：

| 指标 | 当日 | 月累计 | 说明 |
|------|------|--------|------|
| 销售额 | ✅ | ✅ | 销售金额（含税） |
| 毛利额 | ✅ | ✅ | 利润金额 |
| 毛利率 | ✅ | ✅ | 达标判断（≥13%） |
| 订单数 | ✅ | ✅ | 销售订单数量 |
| 亏损订单 | ✅ | ✅ | 毛利率为负的订单 |

**颜色说明**：
- 🟢 绿色 = 达标
- 🔴 红色 = 不达标
- 🟡 黄色 = 预警

### 2. 销售目标进度

**展示内容**：
- 各区域销售汇总
- 每位业务经理的月度目标完成进度
- 当日毛利率和月累计毛利率达标情况

**达标标准**：毛利率 ≥ 13%

### 3. 毛利区间分布

**区间划分**：

| 区间 | 说明 | 目标占比 |
|------|------|---------|
| 亏损(<0%) | 亏本销售 | 0% |
| <5%微利 | 低利润 | ≤10% |
| 5%-8%低利 | 较低利润 | ≤15% |
| 8%-13%基准 | 标准利润 | ~25% |
| 13%-18%中高 | 较高利润 | ~25% |
| ≥18%高利 | 高利润 | ~25% |

## 💡 八、使用技巧

### 1. 每日工作流程

```
早上打开应用
    │
    ├── 📊 查看日报
    │   └── 了解昨日销售、月度进度
    │
    ├── 👥 查看客户
    │   └── 筛选"已超期"客户，优先跟进
    │
    └── 📦 查看产品
        └── 了解库存预警产品
```

### 2. 客户跟进技巧

1. 打开「客户」页面
2. 筛选「归属业务员」= 自己
3. 筛选「预警状态」= 已超期
4. 按超期天数排序，优先跟进超期最久
5. 点击客户查看详情，了解常购产品
6. 联系客户，推荐高毛利产品

### 3. 产品销售技巧

1. 打开「产品」页面
2. 筛选「毛利率」≥ 13%（达标产品）
3. 按销售额排序，找畅销品
4. 查看产品详情，了解采购客户
5. 向同类客户推荐该产品

### 4. 业绩提升技巧

1. 关注「日报」中的毛利率达标情况
2. 多推高毛利产品（≥13%）
3. 减少低毛利和亏损订单
4. 定期跟进超期客户，防止流失

## ❓ 九、常见问题

### Q1：数据什么时候更新？

A：每天早上7:00自动更新前一天的销货数据。

### Q2：为什么毛利率显示不达标？

A：毛利率达标标准是 ≥13%，低于此标准会显示红色。

### Q3：预计下次采购时间怎么算的？

A：系统根据客户历史采购频率自动预测：
```
预计采购日期 = 最近采购日期 + (30天 ÷ 月均采购频次)
```

### Q4：客户预警状态代表什么？

A：表示客户是否超过预计采购时间，超期需要主动跟进。

### Q5：产品覆盖家数是什么？

A：累计采购过该产品的不同客户数量（去重）。

### Q6：离线版本怎么用？

A：下载HTML文件后，用手机浏览器打开即可使用，无需联网。
